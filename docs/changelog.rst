Changelog

0.4
---

0.4.0_
~~~~~~

- Official docker support and docker-compose example
- Autoclean old movies (limit destination)
- Webhooks
- Support for more sources url from `yt-dlp <https://github.com/yt-dlp/yt-dlp>`_
- change downloading program from `youtube_dl <https://youtube-dl.org/>`_ to `yt-dlp <https://github.com/yt-dlp/yt-dlp>`_
- Drop support for python 3.7, 3.8 and 3.9
- fixing documentation

.. warning::
    This version has different configuration file format and terminal client. You can't use

- fix small bugs

- new terminal tool (see: `terminal <content/terminal.html>`_)
- support for all services from `youtube_dl <https://youtube-dl.org/>`_
- locally download url
- add url from console
- auto made new configuration
